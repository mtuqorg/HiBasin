# HiBasin

The hierarchical Bayesian source inversion (HiBASIN) is a python package to perform seismic moment tensor (MT), single force (SF), or joint MT and SF inversion within a hierarchical Bayesian framework incoporating uncertainty estimates for data noise and theory error.

## Citation:
Hu, J., Phạm, T.-S., & Tkalčić, H. (2023). Seismic moment tensor inversion with theory errors from 2-D Earth structure: implications for the 2009–2017 DPRK nuclear blasts. Geophysical Journal International, 235(3), 2035–2054. https://doi.org/10.1093/gji/ggad348
